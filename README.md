# Java-Bank-Simulator
The Java Bank Simulator is a console-based application that mimics a simple banking systems.

It lets users create accounts, deposit, withdraw, transfer money, and view transaction histories — all built using Object-Oriented Programming (OOP) principles.
```
BankAccountSimulator/
│
├── Account.java          # Abstract base class
├── SavingsAccount.java   # Adds interest functionality
├── CheckingAccount.java  # Adds transaction fee logic
├── Transaction.java      # Handles record keeping
├── Bank.java             # Manages all accounts & transfers
└── Main.java             # Console menu interface
```
